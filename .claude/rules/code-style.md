### HTTP ステータスコード

| コード | 使用場面 |
|--------|----------|
| `400` | 業務ルール違反 |
| `401` | トークンなし・期限切れ |
| `403` | ロール制限 |
| `404` | リソース存在しない・スコープ外 |
| `409` | 重複・ロック済み・ステータス不整合 |
| `422` | リクエストボディ不正 |

スコープ外のリソース（他社データ等）は `403` ではなく `404` を返す。

### エラーレスポンス形式

```json
{ "detail": "エラーメッセージ" }
```

バリデーションエラー（422）:
```json
{
  "detail": [
    { "loc": ["body", "field"], "msg": "field required", "type": "missing" }
  ]
}
```
