# メモリ追記ルール

`#` ショートカットや `/memory` で追記するときの判断基準：

- プロジェクト全体の構成・Gotchas → `./CLAUDE.md`
- ルール・規約・方針 → `./.claude/rules/` の該当ファイル
- 自分だけのローカル環境設定 → `./CLAUDE.local.md`
- 全プロジェクト共通の個人設定 → `~/.claude/CLAUDE.md`

## 追記してはいけないもの
- CLAUDE.mdにルール・規約を追加しない（rules/へ）
- 抽象的な指示を書かない。「コードを綺麗に書く」ではなく「インデントはスペース2つ」
